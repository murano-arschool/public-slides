「誰でも作れる」≠「誰でも“いいもの・すごいもの”を作れる」

  • エンジニアに限られていた技術障壁が壊れ、開発が民主化された。
  • すごいもの・いいものを作り切るのはむちゃくちゃ大変。
  • アプリ開発で、より熱意と根性の差が顕著に出る社会に。
